# Java Quiz Console Application

A simple console-based Quiz Application built using Core Java. This project was developed while learning Object-Oriented Programming (OOP) concepts and Java fundamentals.

## Features

* Display multiple-choice quiz questions
* Accept user answers through console input
* Calculate and display the final score
* Object-oriented design using separate classes
* Interactive command-line application

## Technologies Used

* Java
* IntelliJ IDEA

## Project Structure

```text
QUIZ_CONSOLE_APP
├── Main.java
├── Question.java
└── QuestionService.java
```

### Class Responsibilities

#### Main.java

* Entry point of the application
* Creates the QuestionService object
* Starts the quiz and displays the final score

#### Question.java

* Represents a quiz question
* Stores question details, options, and correct answer
* Implements encapsulation using private fields and getter/setter methods

#### QuestionService.java

* Manages quiz questions
* Handles user input
* Evaluates answers
* Calculates and displays the final score

## Concepts Practiced

* Classes and Objects
* Constructors
* Encapsulation
* Arrays of Objects
* Loops
* Conditional Statements
* Method Creation
* User Input using Scanner
* Basic Business Logic Implementation

## Sample Workflow

1. Application starts
2. Quiz questions are displayed
3. User enters answers
4. Answers are evaluated
5. Final score is displayed
